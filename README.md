# VoteWise AI – Smart Election Assistant & Voter Recovery System

**VoteWise AI** is an interactive, AI-powered web application designed to educate Indian citizens on the election process, guide them through their personalized voting journey, and assist in recovering lost voter details securely.

Developed for the **Google Hack2Skill PromptWar Challenge**, this project prioritizes accessibility, security (Zero-PII), and premium visual storytelling.

## 🚀 Key Features

### 1. Immersive "Scrollytelling" Journey
Experience the 5 essential stages of the Indian election lifecycle through a premium, scroll-driven visual narrative:
- **Notification**: Understanding the Election Commission's call to action.
- **Nominations & Campaigns**: Deep dive into candidate transparency.
- **Verification & Inking**: The step-by-step process at the polling booth.
- **Casting Your Vote**: Using the EVM machine with privacy and security.
- **Counting & Results**: The final stage of the democratic process.

### 2. "Know Your Candidate" Deep Research
A brutally realistic candidate explorer that highlights:
- **Criminal Records**: Transparent breakdown of Confirmed Convictions (Red), Pending FIRs (Orange), and Unproven Allegations (Yellow).
- **Financial Disclosures**: Declared assets and liabilities for full transparency.
- **Manifesto Pledges**: Quick, bulleted summaries of candidate promises and past work.

### 3. Smart Voter Slip Recovery (Zero-PII)
A secure simulation tool to help users find their polling booth and serial number using their EPIC number or basic details, without storing any sensitive personal information.

### 4. AI Assistant & Document Verification
A floating AI widget capable of explaining complex election rules and simulating document pre-checks (Aadhaar/Voter ID) to ensure citizens are ready for election day.

## 🎨 Design Philosophy
- **Glassmorphism 2.0**: A premium, high-tech dark mode aesthetic.
- **Behavior-Driven Animation**: Animations designed to guide the user's eye and focus on critical information, inspired by high-end modern web experiences.
- **Accessibility**: High-contrast UI and simplified flows for first-time and elderly voters.

## 🛠️ Technology Stack
- **Frontend**: Vanilla JavaScript, HTML5, CSS3 (Vite)
- **Icons**: Phosphor Icons
- **Deployment**: Firebase Hosting (Google Cloud)
- **Images**: Custom AI-generated 3D Illustrations

## 📦 Getting Started
1. Clone the repository: `git clone https://github.com/Dharmik1817/PromptWar2.git`
2. Install dependencies: `npm install`
3. Run locally: `npm run dev`
4. Build for production: `npm run build`
